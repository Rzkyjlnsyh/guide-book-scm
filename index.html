üîß Troubleshooting

‚ö†Ô∏è Common Issues & Solutions

üö® Issue #1: Negative Stock

Symptoms: Stock qty menunjukkan nilai negatif

Causes: Delivery order dilakukan sebelum stock receipt

Solutions:

  • Check inventory valuation method
  • Verify stock move sequences
  • Adjust inventory dengan physical count
  • Configure negative stock settings
üö® Issue #2: Purchase Order Tidak Bisa Dikonfirmasi

Symptoms: Button "Confirm Order" tidak aktif

Causes: Missing product configuration atau user permissions

Solutions:

  • Check user access rights untuk purchase
  • Verify product configuration (route, type)
  • Ensure vendor information lengkap
  • Check purchase workflow settings
üö® Issue #3: Manufacturing Order Error

Symptoms: Tidak bisa start production

Causes: Insufficient raw materials atau BOM issues

Solutions:

  • Check raw material availability
  • Verify BOM structure dan quantities
  • Run MRP untuk identify shortages
  • Check work center capacity
üö® Issue #4: Sales Order Tidak Generate Delivery

Symptoms: Confirmed sales order tidak create delivery order

Causes: Product route configuration atau stock availability

Solutions:

  • Check product routes (MTO/MTS)
  • Verify warehouse configuration
  • Check stock availability
  • Review delivery order creation rules
